# DataBase System<br/>
MySQL과 spring 프레임워크를 사용하여 구축한 학생 관리 시스템이다.<br/>
학생 정보 관리 시스템의 주요 기능으로는 회원가입, 로그인, 공지사항 확인, 수강신청 등이 있다.<br/>
시스템에 구현된 여러 기능을 통해 학생 별 개인 정보를 데이터베이스에 저장한다.<br/><br/><br/>
